1. Introduction

RoLookup v3 ("the Bot") is a Discord bot that provides Roblox account lookup services. This Privacy Policy explains what data we collect, how we use it, and your rights regarding your personal information.

By using the Bot, you agree to the collection and use of information as described in this policy.

2. Information We Collect

2.1 Discord User Data

  • Discord User ID - Used to link your Discord account to Roblox accounts
  • Server/Guild ID - For server-specific configurations

2.2 Roblox User Data

  • Roblox User ID - Unique identifier for Roblox accounts
  • Roblox Username - Current Roblox display name
  • Roblox Profile URL - Generated link to the Roblox profile

2.3 Server Configuration Data

Server administrators can configure the following, which we store:

  • Custom Command Prefixes - Custom triggers for bot commands
  • Ignored Channels - Channel IDs where commands are disabled
  • Whitelisted Roles - Role IDs that have command access
  • Update Channels - Channel IDs for bot update announcements
  • Update Roles - Role IDs to ping for updates

2.4 Privacy Settings

  • Privacy Mode Preference - Whether you've enabled auto-deletion of command messages

2.5 Automatically Collected Data

  • Command Usage Logs - Anonymous logs containing timestamps and command types only (no user identifiers). Used solely for error debugging and troubleshooting.
  • Error Logs - Anonymous technical information (e.g., error messages, stack traces) collected only when errors occur. No personal data is included.

3. How We Use Your Information

We use the collected data for the following purposes:

  • Account Linking - To associate Discord accounts with Roblox accounts for quick lookups
  • Command Execution - To provide requested Roblox user information
  • Server Configuration - To respect administrator settings (prefixes, ignored channels, role restrictions)
  • Privacy Mode - To automatically delete command messages when enabled
  • Update Broadcasts - To notify servers about bot updates (if configured)
  • Error Resolution - To debug and fix technical issues

We do not sell, rent, or trade your personal information with third parties.

4. Data Storage and Retention

4.1 Storage Method

All data is stored in a SQLite database on our secure servers.

4.2 Retention Periods

Data TypeRetention Period
Discord-Roblox LinksIndefinite (until manually removed)
Privacy SettingsIndefinite (until changed by user)
Server ConfigurationsIndefinite (until server leaves or admin removes)
Command LogsIndefinite (stored anonymously)

4.3 Data Security

We implement reasonable security measures to protect your data, including secure storage of database files and restricted access to server infrastructure.

5. Third-Party Services

The Bot interacts with the following third-party services to provide its functionality:

5.1 Roblox Corporation APIs

We use official Roblox APIs to fetch user information including username, groups, outfits, and account details.

  • Data Shared: Roblox User IDs, usernames
  • Purpose: Fetching Roblox account information for lookups

5.2 Rolimons API

We use Rolimons to fetch limited item values and player inventories.

  • Data Shared: Roblox User IDs, item IDs
  • Purpose: Providing item values and inventory information

5.3 Hermivore API (Optional)

We may use Hermivore for Discord-Roblox account linking.

  • Data Shared: Discord User IDs, Roblox authentication data
  • Purpose: Verifying Discord-Roblox account ownership

5.4 Self-Hosted API (Optional)

We may use a self-hosted API service for account linking.

  • Data Shared: Discord User IDs, Roblox authentication data
  • Purpose: Verifying Discord-Roblox account ownership

5.5 BloxLink Integration (Optional)

We may use BloxLink's channel-based verification system.

  • Data Shared: Discord User IDs
  • Purpose: Alternative method for verifying Discord-Roblox account ownership

6. Your Rights

6.1 Access to Your Data

You may request a copy of your personal data by contacting the bot developers (see Section 10).

6.2 Data Deletion

  • Discord-Roblox Links: There is no user-accessible command to delete linked accounts. Contact bot developers for manual removal.
  • Privacy Settings: Use the .privacy command to toggle your privacy preference.
  • Server Configurations: Server administrators can remove configurations using the following commands:
    • .setprefix reset - Remove custom prefix
    • .unignorechannel - Remove channel ignore
    • .unwhitelistrole - Remove role whitelist
    • .removeupdates - Remove update channel configuration

6.3 Opt-Out Rights

  • You can prevent the Bot from processing your commands by not using the Bot
  • Server administrators can ignore channels or whitelist roles to restrict Bot access
  • You can enable privacy mode (.privacy) to auto-delete command messages

7. Children's Privacy

The Bot is not intended for users under the age of 13. We do not knowingly collect personal information from children under 13. If we become aware that we have collected data from a child under 13, we will take steps to delete it promptly.

8. Changes to This Privacy Policy

We encourage users to review this policy periodically.

9. Data Security
