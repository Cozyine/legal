  • Discord User ID - Links your Discord to Roblox accounts

2. How We Use Data

  • Link Discord accounts to Roblox accounts
  • Respond to lookup commands
  • Respect server configurations
  • Debug errors when they occur

3. Data Storage

  • Stored in SQLite database on secure servers
  • Discord-Roblox links kept until manually removed
  • Server configs kept until server leaves
  • Logs stored anonymously indefinitely
